About “live” results

This app reads data/draws.json. To make results live, run the scrapers in scrapers/ on a schedule (e.g. the included GitHub Actions template in ops/) so the file is refreshed after every draw (Wed / Sat / Sun ~7:00 PM MYT, plus special Tuesday draws). The service worker then picks up the new data automatically.

Digit frequency by position

Longest absent numbers (cold / overdue)

Numbers that have not appeared for the longest time in the selected scope. “Overdue” is a description of the past, not a prediction — every draw is independent.

When did a number win?

Enter any 4-digit number to list every draw where it won, with date, operator and prize tier.

Number analyzer

Statistical profile of your number against the full history: empirical frequency, gaps, weekday pattern and the theoretical odds.

Do weekdays matter? (χ² test)

For each draw day, we test whether the digit distribution in winning numbers differs from uniform (the fair-lottery hypothesis). A p-value above 0.05 means no evidence of weekday bias.

Digit frequency by weekday

Empirical per-position digit frequencies, split by draw day. This is the data the naive-Bayes weekday score in the Analyzer is built on.

“Most likely” digits per position (per selected weekday)

The maximum-likelihood digits under the weekday model. Shown for education — in a fair lottery these edges are statistical noise and do not persist.

About this app

Malaysia 4D Insights is an installable, offline-capable PWA for exploring results of the three national 4D operators: Magnum 4D, Da Ma Cai 1+3D and Sports Toto 4D.

Data pipeline

scrapers/ contains reference Python scrapers for the operators’ public result pages and ops/update-results.yml is a GitHub Actions template that refreshes data/draws.json after every draw. If a source only exposes the top prizes, the app keeps working — every statistic has a “Top 3 prizes only” mode.

Honesty box

A fair 4D draw is uniform and independent: every number has probability 1/10,000 of taking first prize, every draw, regardless of history. The statistics here describe the past; the χ² page tests (and with real data, generally confirms) that past frequencies carry no predictive power. Nothing in this app improves your odds of winning.
